---
title: 노티플라이 개발자 가이드
---

노티플라이는 유저에게 앱 푸시, 팝업, 카카오톡, 이메일 등 다양한 채널의 메시지를 쉽고 빠르게 발송하고 자동화할 수 있도록 돕는 마케팅 솔루션입니다. 유저 행동에 따라 개인화된 메시지를 실시간으로 보내고, 메시지 캠페인의 성과를 측정할 수 있는 기능을 제공합니다.
개발자 가이드에서 모바일 및 웹 개발자가 서비스에 노티플라이를 연동 개발하기 위한 내용을 확인해 보세요.

---

## 기본 정보

| 항목 | 값 |
|---|---|
| 지원 플랫폼 | iOS, Android, React Native, Flutter, JavaScript |
| 기반 기술 | Firebase Cloud Messaging (FCM) |
| 주요 기능 | 푸시 알림, 인앱 팝업, 유저 트래킹, 세그먼테이션 |
| 자동 수집 데이터 | 유저 세션, 기기 정보, 푸시 토큰 |

### 필수 사전 작업

> **중요**: 모든 SDK는 Firebase Cloud Messaging(FCM)을 기반으로 동작합니다.  
> SDK를 설치하기 전에 반드시 **Firebase 프로젝트를 생성하고 노티플라이와 연동**해야 합니다.

---

## 빠른 시작 가이드

<Steps>
  <Step title="Firebase 연동">
    먼저 Firebase 프로젝트를 생성하고 노티플라이 콘솔과 연동합니다.
  </Step>
  <Step title="플랫폼별 SDK 설치">
    원하는 플랫폼의 SDK를 선택하여 설치합니다.
  </Step>
  <Step title="기본 설정">
    SDK 초기화 및 유저 식별자 설정을 완료합니다.
  </Step>
  <Step title="테스트">
    푸시 알림 및 인앱 팝업이 정상 작동하는지 확인합니다.
  </Step>
</Steps>
