## 개요

카카오 브랜드 메시지는 카카오 브랜드 채널로부터 커뮤니케이션 수신에 동의한 사용자에게 마케팅 콘텐츠를 보내는 프리미엄 메시징 채널입니다. 리치 미디어 형식을 제공하며 사용자의 명시적인 마케팅 동의가 필요합니다.

## 작동 방식

1. 사용자가 카카오 브랜드 채널에 전화번호를 등록합니다
2. 사용자가 마케팅 메시지 수신에 동의합니다
3. 템플릿을 사용해 메시지를 작성합니다
4. 동의한 사용자에게 메시지가 발송됩니다
5. 발송 실패 시 SMS로 대체 발송 가능

## 요구 사항

- 사용자가 카카오 브랜드 채널에 전화번호 등록
- 사용자가 마케팅 동의
- 카카오 브랜드 채널 구성
- 승인된 메시지 템플릿

## 타겟 유형

| 타겟 | 한글 | 수신자 |
|--------|--------|--------------|
| Marketing Agreement | M (마수동) | 광고 수신 동의한 사용자 |
| Marketing + Channel Friend | N | 마케팅 동의 + 채널 친구 |
| Marketing AND Channel Friend | I | 둘 다 있어야 함 |

## 메시지 유형

| 유형 | 설명 |
|------|-------------|
| Text | 단순 텍스트 메시지 |
| Image | 이미지 포함 메시지 |
| Wide image | 전체 너비 이미지 |
| Wide item list | 목록이 있는 이미지 |
| Carousel | 스와이프 가능한 카드 |
| Premium video | 비디오 콘텐츠 |
| Commerce | 상품 쇼케이스 |
| Carousel commerce | 여러 상품 |

## 메시지 콘텐츠

| 필드 | 설명 | 필수 |
|-------|-------------|----------|
| Template code | 승인된 템플릿 ID | 예 |
| Type | 메시지 형식 | 예 |
| Template params | 템플릿용 값 | 예 |
| Image params | 이미지 URL | 이미지 유형용 |
| Video params | 비디오 URL 및 썸네일 | 비디오 유형용 |
| Failover SMS | 백업 메시지 | 아니오 |

## 템플릿 파라미터

템플릿 변수 채우기:
- 텍스트 값 (문자열)
- 숫자
- 불리언 (true/false)
- 선택 사항이면 null 가능

## 이미지 파라미터

이미지 기반 메시지용:
- Image URL (필수)
- Image link (선택, 탭했을 때 이동할 위치)

## 비디오 파라미터

프리미엄 비디오 메시지용:
- Video URL (필수)
- Thumbnail URL (선택, 미리보기 이미지)

## 개인화

카카오 브랜드 메시지는 [Liquid 템플릿](/ko/knowledge-base/concepts/liquid-templates) 개인화를 전체 지원합니다.

**지원되는 필드:**
- 템플릿 파라미터
- 대체 발송 SMS 텍스트

**예시:**
```liquid
Template param (customer_name): {{user.$name | default: "고객"}}
Template param (points_balance): {{user.loyalty_points | default: 0}}
Failover SMS: {{user.$name}}님, 현재 포인트 잔액은 {{user.loyalty_points}}점입니다.
```

**참고:** 알림톡처럼 템플릿 구조는 사전 승인되어 있지만, 파라미터 값은 개인화할 수 있습니다.

## SMS 대체 발송

브랜드 메시지 발송이 실패하면:
- 사용자 미등록
- 사용자 동의 철회
- 자동으로 백업 SMS 발송 (구성된 경우)

## 규칙

- 사용자가 마케팅 동의를 해야 함
- 템플릿은 승인받아야 함
- 리치 콘텐츠는 형식 요구 사항이 있음
- [피로도 관리](/ko/knowledge-base/rules/fatigue-management) 적용
- [제외 사용자](/ko/knowledge-base/rules/excluded-users)에서 비활성화 가능

## 예외 사항

- 마케팅 동의 없는 사용자는 메시지 수신 불가
- 템플릿 파라미터는 예상 형식과 일치해야 함
- 대용량 미디어 파일은 발송 실패할 수 있음
