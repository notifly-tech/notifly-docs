## 개요

Notifly는 사용자와 그들의 기기를 추적합니다. 한 사용자가 여러 기기를 가질 수 있습니다 (휴대폰과 태블릿처럼). 이는 올바른 곳에 메시지를 보내고 콘텐츠를 개인화하는 데 도움이 됩니다.

## 사용자 (Users)

사용자는 앱의 한 사람을 나타냅니다.

### 사용자 식별자

| ID | 설명 |
|----|------------|
| Notifly User ID | Notifly가 생성한 고유 ID (항상 존재) |
| External User ID | 사용자를 위한 고유 ID (선택 사항, 계정 ID 같은 것) |

### 사용자 속성

각 사용자에 대해 저장되는 정보:

| 속성 | 예시 |
|----------|---------|
| Email | user@example.com |
| Phone number | +82-10-1234-5678 |
| User properties | 모든 커스텀 데이터 (이름, 선호도 등) |
| Timezone | Asia/Seoul (IANA 형식) |
| Random bucket number | 0-99 (A/B 테스트용) |

**참고:** 사용자 타임존은 [로컬 타임존 발송](/ko/knowledge-base/concepts/local-timezone-delivery)에 사용됩니다. 메시지 발송 시점을 결정할 때 기기 타임존보다 우선합니다.

### 사용자 출처

사용자가 어디서 왔는지:
- SDK 연동
- CSV 업로드
- API 가져오기
- 이커머스 연동 (Cafe24 등)

## 기기 (Devices)

기기는 하나의 휴대폰, 태블릿, 또는 컴퓨터입니다.

### 기기 식별자

| ID | 설명 |
|----|------------|
| Notifly Device ID | Notifly가 생성한 고유 ID |
| Device Token | 푸시 알림을 보내기 위한 토큰 |

### 기기 속성

| 속성 | 설명 |
|----------|------------|
| Platform | ios, android, 또는 web |
| OS Version | 예: "iOS 17.2" 또는 "Android 14" |
| App Version | 앱의 버전 번호 |
| SDK Version | Notifly SDK 버전 |
| Timezone | 기기 타임존 (IANA 형식, 예: "America/New_York") |
| Device properties | 기기에 대한 커스텀 데이터 |

**참고:** 기기 타임존은 사용자의 타임존이 설정되지 않은 경우 [로컬 타임존 발송](/ko/knowledge-base/concepts/local-timezone-delivery)의 대체값으로 사용됩니다.

### 푸시 알림 권한

| 상태 | 의미 |
|--------|---------------|
| Not Determined | 사용자에게 아직 물어보지 않음 |
| Denied | 사용자가 알림을 거부함 |
| Authorized | 사용자가 알림을 허용함 |
| Provisional | 조용한 알림 허용됨 (iOS) |

## 사용자-기기 관계

```
사용자 (사람)
├── 기기 1 (iPhone)
├── 기기 2 (iPad)
└── 기기 3 (웹 브라우저)
```

- 한 사용자가 여러 기기를 가질 수 있음
- 각 기기는 한 사용자에게 속함
- 사용자 ID를 설정하면 모든 기기가 함께 연결됨

## LINE 사용자

LINE 메시징의 경우 추가 정보가 있습니다:

| 속성 | 의미 |
|----------|---------------|
| LINE User ID | 사용자의 LINE 식별자 |
| Following Status | LINE 계정을 팔로우하고 있나요? |

팔로우 상태:
- **Followed** - 사용자가 친구로 추가함
- **Unfollowed** - 사용자가 차단함
- **Unknown** - 상태를 확인할 수 없음

## 규칙

- 기기가 없는 사용자도 이메일/SMS를 받을 수 있음
- 푸시 알림은 유효한 기기 토큰이 필요함
- 사용자가 로그아웃하면 기기가 업데이트되어야 함
- 기기 토큰은 만료되거나 무효화될 수 있음
