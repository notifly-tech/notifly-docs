## 개요

Notifly 콘솔은 프로젝트를 관리하는 곳입니다. 프로젝트 설정에서 프로젝트 작동 방식, 접근 권한, 보안 기능 활성화를 구성할 수 있습니다.

## 설정 섹션

### 보안 설정

프로젝트의 접근과 인증을 제어합니다.

- [보안 설정](/ko/knowledge-base/console/security-settings) - 사용자 권한, 세션, OTP, IP 제한

### 추후 추가 예정

- 사용자 관리 - 팀원 초대 및 관리
- API 설정 - API 키와 웹훅 관리
- 알림 설정 - 관리자 이메일 알림
- 프로젝트 정보 - 기본 프로젝트 세부 사항 및 설정

---

## 접근 방법

1. Notifly 콘솔에 로그인
2. 프로젝트 클릭
3. 사이드바에서 **설정** (톱니바퀴 아이콘) 이동
4. 구성하려는 설정 섹션 선택
