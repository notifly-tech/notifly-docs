## 개요

Naver Commerce (스마트스토어라고도 함)는 네이버의 이커머스 플랫폼입니다. 많은 한국 기업들이 네이버를 통해 판매합니다. Notifly와 연동하면 이 고객들에게 타겟 메시지를 발송할 수 있습니다.

## 작동 방식

1. 네이버 스마트스토어를 Notifly에 연결
2. 고객 및 주문 데이터 동기화
3. 쇼핑 이벤트 추적
4. 구매 행동 기반 campaign 구축

## 동기화되는 데이터

### 고객 정보
- 고객 식별자
- 연락처 정보 (가능한 경우)
- 구매 이력

### 주문 데이터
- 주문 상세
- 상품 정보
- 주문 상태 업데이트

## 사용 가능한 이벤트

| 이벤트 | 발생 시점 |
|--------|----------|
| Order placed | 고객이 구매 완료 |
| Order status changed | 배송, 전달 업데이트 |
| Review requested | 배송 완료 후 |

## 활용 사례

| 활용 사례 | 방법 |
|----------|------|
| 주문 확인 | order_placed에서 트리거 |
| 배송 업데이트 | status_changed에서 트리거 |
| 리뷰 요청 | 배송 후 지연 |
| 재구매 유도 | Segment: X일 전 구매 |
| 크로스셀 | 구매 이력 기반 |

## 메시지 발송 옵션

### 카카오 연동
많은 네이버 쇼핑객이 카카오를 사용하므로:
- 주문 업데이트에 [카카오 알림톡](/ko/knowledge-base/channels/kakao-alimtalk) 사용
- 마케팅에 [카카오 친구톡](/ko/knowledge-base/channels/kakao-friendtalk) 사용
- 카카오 미사용자를 위한 SMS fallback

### SMS
- 직접 문자 메시지
- 모든 한국 전화번호에서 작동

## Segment

다음을 사용하여 segment 구성 가능:
- 구매 이력
- 주문 금액
- 상품 카테고리
- 고객 최근성

## 규칙

- 네이버 스마트스토어 연결 필요
- 네이버 정책 준수 필수
- 마케팅에 고객 동의 필요
- 한국 개인정보보호법 적용

## Cafe24와의 차이점

| 항목 | Naver Commerce | Cafe24 |
|------|----------------|--------|
| 플랫폼 | 마켓플레이스 | 호스팅 스토어 |
| 데이터 접근 | API 통한 | 직접 연동 |
| 앱 푸시 | 네이티브 앱 없음 | Cafe24 앱 가능 |
