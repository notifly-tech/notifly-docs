## 긴급 빠른 참조

### "왜 내 메시지가 발송되지 않았나요?"

다음 순서로 확인하세요:

| 확인 항목 | 확인 방법 | 해결책 |
|----------|----------|--------|
| 1. Campaign 활성 상태? | Campaign 상태 = "Active" | Campaign 활성화 |
| 2. 사용자가 segment에 있나? | Segment 조건 확인 | Segment 규칙 조정 |
| 3. 사용자가 제외됐나? | 제외 사용자 목록 확인 | 제외 목록에서 제거 |
| 4. 발송 금지 시간? | Forbidden timing 설정 확인 | 대기 또는 타이밍 조정 |
| 5. 피로도 한도 도달? | Campaign fatigue 설정 확인 | 리셋 또는 한도 증가 |
| 6. 전역 한도 도달? | 프로젝트 frequency 설정 확인 | 대기 또는 한도 증가 |
| 7. 채널 요구사항 충족? | 디바이스 토큰, 이메일, 전화번호 존재 여부 | 필수 데이터 확보 |

전체 의사결정 트리는 [메시지가 발송되지 않음](/ko/knowledge-base/troubleshooting/message-not-sending)을 참조하세요.

### "왜 사용자가 내 저니에 진입하지 않나요?"

| 확인 항목 | 문제 |
|----------|------|
| Journey 상태 | "Running"이어야 함 |
| 진입 트리거 | 이벤트가 정확히 일치해야 함 |
| 진입 segment | 사용자가 segment에 일치해야 함 |
| 테스트 모드 | 화이트리스트 사용자만 진입 |
| 재진입 설정 | 사용자가 이미 완료했을 수 있음 |

자세한 내용은 [저니 디버깅](/ko/knowledge-base/troubleshooting/journey-debugging)을 참조하세요.
