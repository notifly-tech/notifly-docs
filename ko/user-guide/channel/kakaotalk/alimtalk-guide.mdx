## 개요

노티플라이에서는 유저들에게 카카오 알림톡을 원하는 타이밍에 보낼 수 있습니다.

- 노티플라이에서는 타 발송 대행사 없이 직접 카카오 알림톡 발송을 하실 수 있습니다.
- 카카오톡 비즈메시지(알림톡, 브랜드메시지)를 발송하려면 카카오톡 채널을 먼저 등록해야 합니다. 카카오톡 채널은 [카카오톡 채널 등록 페이지](https://center-pf.kakao.com)에서 무료로 만들 수 있습니다.
- 카카오톡 비즈메시지 발송을 위해서는 비즈니스 채널로의 전환이 필수입니다. 자세한 정보는 [카카오 비즈니스 가이드](https://kakaobusiness.gitbook.io/main/channel/start)에서 확인해주세요.
- 카카오 알림톡은 **템플릿 기반으로 발송됩니다**. 템플릿은 검수 후 발송 가능하며, 검수 기간은 대략 1~3일정도 소요됩니다. 자세한 내용은 [카카오톡 템플릿 심사 FAQ](https://cs.kakao.com/helps?service=159&category=523&locale=ko)를 참고해 주세요.

## 1. 카카오 알림톡 발신 채널 등록하기

[카카오 비즈메시지 발송 환경 셋팅 방법](/ko/user-guide/channel/kakaotalk/setup)을 참고하여 채널을 노티플라이에 등록해 주세요.

## 2. 카카오 알림톡 템플릿 심사받기

카카오 알림톡은 카카오의 **사전 승인을 받은 템플릿 기반으로 발송**됩니다. 캠페인을 집행하기 전 메시지에 이용할 템플릿을 생성하고 사전 심사를 받으세요.

![Alimtalk Template Pn](/images/alimtalk_template.png)

1. '카카오 템플릿' \> '알림톡 템플릿'에서 '**\+ 새 템플릿 생성하기**'를 클릭합니다.
2. 상단 드롭다운에서 어느 카카오 채널의 알림톡 템플릿을 만들 것인지 선택합니다.
3. 템플릿 이름을 작성합니다. 이 정보는 운영 시 식별하기 위한 이름으로 유저에게 노출되지 않습니다.
4. 템플릿 유형을 선택하고 메시지 내용을 작성합니다. 재심사 과정을 생략하기 위해 유형별 작성 가이드를 반드시 숙지해주세요.
   1. 템플릿에는 치환 가능한 변수 필드를 추가할 수 있습니다. '**\+ 변수 추가**' 혹은 '**URL 개인화**' 버튼을 눌러 변수가 들어가야 할 위치를 정합니다. 잘 작성되면 `#{변수}`와 같은 형식으로 삽입됩니다.
1. '저장' 버튼을 눌러 저장합니다.

심사는 영업일 기준 1~3일 소요되며 반려 시 심사기관에서 남긴 코멘트를 통해 미비사항을 확인하고 보완하여 재심사 요청할 수 있습니다.

## 3. 카카오 알림톡 캠페인 시작하기

노티플라이 캠페인 생성 페이지에서 카카오 알림톡을 선택해 주세요.

<Info>
  카카오 알림톡에서 **휴대폰 번호** 정보는 발송에 필요한 필수 요소입니다. 휴대폰 번호를 불러오는 원천 소스는 발송 대상 종류마다 상이할 수 있습니다.
</Info>

### 3-1. 발송 대상 설정

카카오 알림톡의 경우, 지원되는 발송 대상 종류는 다음과 같습니다:

1. 유저 그룹 설정
2. CSV 파일 업로드 3. 유저ID 기반 3. 전화번호 기반

#### 유저 그룹 설정

자세한 내용은 [유저 그룹 설정](/ko/user-guide/campaigns/campaign-segments/segment#1-유저-그룹-설정) 섹션을 참고해주세요.

#### CSV 파일

노티플라이 데이터베이스에 등록되지 않은 유저에게도 직접 발송하고 싶다면, CSV 파일 업로드―전화번호 기반― 기능을 활용할 수 있습니다. 자세한 내용은 [발송 대상 설정 - 2. CSV 업로드](/ko/user-guide/campaigns/campaign-segments/segment#2-csv-업로드) 섹션을 참고해주세요.

- **`카카오톡 발송에서 phone_number 컬럼은 필수이며 중복되거나 비어있으면 안됩니다.`**

등록이 잘 되었다면 다음과 같이 발송 대상과 등록된 필드명이 표시됩니다.

![Kakao alimtalk csv upload result](/ko/user-guide/images/user-guide/kakaotalk/ko/assets/images/kakao_alimtalk_csv_upload_result-1ced433b6b74e71bc0d72a0df342f609.png)

### 3-2. 메시지

카카오 알림톡은 **템플릿 기반으로 발송**됩니다. 템플릿 생성 시 치환 가능한 변수 필드의 위치를 사전에 지정할 수 있으며, `#{변수}`와 같은 형식으로 되어있습니다. 해당 변수는 특정 값을 직접 입력하여 치환할 수 있으며, [메시지 개인화](/ko/user-guide/campaigns/personalization/basic)를 통하여 유저 정보를 활용해 개인에게 다르게 메시지를 보낼 수도 있습니다.

1. <u>유저 그룹 설정</u>을 이용하여 대상을 지정할 때에는, **모든 변수 필드에 대하여 치환할 문구**를 입력하셔야 합니다. 치환 문구를 입력하실 때에 개인화가 필요하다면 항목 하단 '**개인화 변수**' 버튼을 눌러서 개인화 문구를 추가해주세요. 필요하지 않다면 고정된 특정 값을 입력해 주세요. 메시지 개인화에 대한 자세한 정보는 [메시지 개인화](/ko/user-guide/campaigns/personalization/basic) 섹션을 참고해주세요.

![image.png](/images/image.png)

2. CSV 업로드 기능을 이용한 캠페인인 경우, **CSV 필드명과 카카오톡 템플릿 변수의 이름이 같을 경우 자동으로 매칭됩니다.** CSV에 작성하지 않은 필드는 직접 입력해주세요.
3. '**실패 시 대체 문자 메시지**' 토글을 활성화하여 알림톡 발송 실패 시 대체로 발송할 문자메시지를 세팅할 수 있습니다. (Pro plan 이상)

   <Info>
     알림톡은 정보성 메시지이기 때문에 080 무료 수신 거부 필터가 작동하지 않습니다.
   </Info>
