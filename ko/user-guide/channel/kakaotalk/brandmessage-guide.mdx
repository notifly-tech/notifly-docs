## 1. 브랜드 메시지 타깃팅 이해하기

카카오 브랜드 메시지는 마케팅 정보 수신동의 여부와 카카오 채널 친구 여부를 이용해 다양한 타겟 그룹을 지정할 수 있습니다.

![Brandmessge Group](/images/brandmessge_group.png)

### 채널 친구 마수동 그룹 (I)

- 카카오 채널 친구이면서 마케팅 정보 수신에 동의한 유저를 타겟합니다.

### 마수동 그룹 (M)

- 카카오 채널 친구 여부와 관계 없이 마케팅 정보 수신에 동의한 유저를 타겟합니다.

### 비친구 마수동 그룹 (N)

- 카카오 채널 친구가 아니면서 마케팅 정보 수신에 동의한 유저를 타겟합니다.

<Info>
  노티플라이는 휴대전화번호 정보가 수집된 유저에게만 브랜드메시지를 발송할 수 있습니다. 휴대 전화번호 정보가 수집되지 않은 채널 친구를 타깃한 메시지 발송은 카카오 모먼트를 이용해주세요.
</Info>

<Info>
  M그룹과 N그룹은 채널 친구가 아닌 유저가 포함된 그룹으로, '비친구 그룹'으로 구분됩니다.
</Info>

<Info>
  브랜드 메시지는 광고성 메시지 채널이기 때문에 야간 시간에는 발송이 불가합니다.
</Info>

## 2. 카카오 브랜드 메시지 채널 등록하기

### 2-1. 카카오 채널 개설하기

[**카카오 채널 관리자 센터**](https://business.kakao.com/info/kakaotalkchannel/)를 통해 직접 개설 신청을 해주세요. 카카오톡 채널 개설 후 비즈니스 채널로 전환이 필요하며, 홈공개 ON 상태여야 합니다.

### 2-2. 노티플라이에 카카오 채널 연결하기

[**카카오 비즈메시지 발송 환경 셋팅 방법**](https://docs.notifly.tech/ko/user-guide/channel/kakaotalk/setup)을 참고하여 채널을 노티플라이에 등록해 주세요.

### 2-3. 브랜드 메시지 비친구 그룹(M, N) 메시지 발송 사전 신청 (옵션)

브랜드 메시지를 비친구 그룹(M, N)에게 발송하기 위해서는 카카오에 사전 신청을 해야합니다.

- 신청을 원하시는 경우 👉[[노티플라이] 브랜드 메시지 비친구(M,N) 발송 신청서](https://tally.so/r/xXjMb5)를 제출해주세요.

  <Accordion title="비친구 메시지 발송 (타겟팅 M, N) 사용 가능한 조건" icon="sparkles">
    - 비즈니스 인증받은 채널
    - 등록된 채널에 전화번호가 등록되어 있음 (카카오 비즈니스 파트너센터 채널 \> 채널 \> 채널 정보 \> 채널 정보 내 대표 전화번호 or 고객센터 전화번호 입력 필요)
    - 채널 친구수 5만 이상
    - 3개월 이내 알림톡 발송이력 존재
    - **광고성 정보 수신동의 증적파일**
    - **080 무료수신거부 전화번호**
  </Accordion>

<Info>
  타딜러사에서 업로드한 파일이 이미 존재하는 경우 파일 업로드 과정을 생략하고 비친구 메시지 발송 (타겟팅 M, N) 사용 신청이 가능합니다.
</Info>

<Check>
  광고성 정보 수신동의 증적 자료 예시

  - 서비스 내 마케팅 수신동의를 유저가 선택할 수 있는 설정페이지 스크린샷
</Check>

## 3. 브랜드 메시지 캠페인 시작하기

### 3-1. 브랜드 메시지 템플릿 생성하기

카카오 브랜드 메시지는 **템플릿 기반으로 발송됩니다**. 템플릿을 먼저 작성해주세요.

1. 좌측 메뉴 '카카오 템플릿' \> '브랜드 메시지 템플릿'에서 '**\+ 새 템플릿 생성**'를 클릭합니다.
2. 템플릿 이름을 입력하고 템플릿을 등록할 카카오 채널을 선택합니다.
3. 발송할 유형을 선택하고 본문, 이미지, 버튼 등 내용을 작성합니다. 총 8개 유형의 규격 사항은 아래 링크를 참고해주세요.

   <Accordion title="브랜드 메시지 유형 종류" icon="sparkles">
     - [텍스트형](https://kakaobusiness.gitbook.io/main/ad/brandmessage#undefined-1)
     - [이미지형](https://kakaobusiness.gitbook.io/main/ad/brandmessage#undefined-2)
     - [와이드 이미지형](https://kakaobusiness.gitbook.io/main/ad/brandmessage#undefined-3)
     - [와이드 아이템리스트형](https://kakaobusiness.gitbook.io/main/ad/brandmessage#undefined-4)
     - [캐러셀 피드형](https://kakaobusiness.gitbook.io/main/ad/brandmessage#undefined-6)
     - [커머스형](https://kakaobusiness.gitbook.io/main/ad/brandmessage#undefined-8)
     - [캐러셀 커머스형](https://kakaobusiness.gitbook.io/main/ad/brandmessage#undefined-9)
     - [프리미엄 동영상형](https://kakaobusiness.gitbook.io/main/ad/brandmessage#undefined-7)
   </Accordion>
   - 템플릿에는 개인화 변수 필드를 추가할 수 있습니다. '**\+ 변수 추가**' 버튼을 눌러 변수가 들어가야 할 위치를 정합니다. 잘 작성되면 `#{변수}`와 같은 형식으로 삽입됩니다.
4. 템플릿을 저장합니다.

### 3-2. 캠페인 세팅하기

노티플라이 캠페인 생성 페이지에서 브랜드 메시지를 선택해 주세요.

### 발송 대상 그룹 설정

노티플라이로 선택할 수 있는 브랜드 메시지 타깃 그룹은 다음과 같습니다. 알맞은 발송 대상을 선택해주세요.

![Brand Message Targetgroup](/images/brand_message_targetgroup.png)

- **M: 마케팅 수신 동의 유저** = 우리 서비스 유저 모두
- **N**: **친구가 아닌 마케팅 수신 동의 유저**  = 우리 서비스 유저 중 채널 친구가 아닌 유저
- **I: 채널 친구** = 우리 서비스 유저이면서 채널 친구인 경우

### 유저 그룹 설정

자세한 내용은 [유저 그룹 설정](https://docs.notifly.tech/ko/user-guide/campaigns/campaign-segments/segment#1-%EC%9C%A0%EC%A0%80-%EA%B7%B8%EB%A3%B9-%EC%84%A4%EC%A0%95) 섹션을 참고해주세요.

<Info>
  카카오 브랜드 메시지에서 **휴대폰 번호 정보**는 유저를 타게팅하여 발송에 필요한 필수 요소입니다.
</Info>

<Check>
  비친구(M,N)그룹 발송 시 **마케팅 정보 수신 동의 여부에 동의한 유저만을 필터**하였는지 확인해주세요.
</Check>

### 메시지 및 액션

다음으로 템플릿을 선택하고 변수값를 세팅합니다.

1. 앞서 생성한 템플릿을 등록합니다. 템플릿 관리는 ‘카카오 템플릿’ \> ‘브랜드 메시지 템플릿’을 참고해주세요.
4. 변수가 있는 템플릿의 경우:
   1. **유저 그룹 설정**을 이용하여 대상을 지정할 때에는, **모든 변수 필드에 대하여 발송될 메시지에 적용될 값**를 세팅해야 합니다. 개인화가 필요하다면 항목 하단 '**개인화 변수**' 버튼을 눌러서 개인화 문구를 추가해주세요. 필요하지 않다면 고정된 특정 값을 입력해 주세요. 메시지 개인화에 대한 자세한 정보는 [메시지 개인화](/ko/user-guide/campaigns/personalization/basic) 섹션을 참고해주세요.
   1. **파일 업로드**를 이용하여 대상을 지정할 때에는, **파일 필드명과 카카오톡 템플릿 변수의 이름이 같을 경우 자동으로 매칭됩니다.** 파일에 작성하지 않은 필드는 직접 입력해주세요.
1. 필요한 경우, ‘실패 시 대체 문자 메시지’를 세팅해주세요. 브랜드 메시지 발송 실패 시 문자메시지로 대체 발송 합니다.

<Info>
  예상 모수 계산 시 세팅된 유저 조건을 만족하면서 휴대 전화번호 정보가 있는지 여부를 확인하여 집계합니다.
</Info>

## FAQ

<AccordionGroup>
  <Accordion title="유저 조건으로 세팅된 유저 중 일부가 선택한 발송 대상 그룹이 아니면 어떻게 발송되나요?">
    예를 들어 발송대상에서 **N**: **친구가 아닌 마케팅 수신 동의 유저**를 선택하고 유저 그룹 조건으로 **I: 채널 친구**에 해당하는 유저에게 발송시도를 한다면 메시지는 발송이 되지 않습니다. 이 경우 **M: 마케팅 수신 동의 유저** 혹은 **I: 채널 친구**로 발송 대상 그룹을 선택해주셔야 합니다.
  </Accordion>
  <Accordion title="메시지를 수신하지 못했습니다. 무엇을 확인해야 하나요?">
    다음 내용을 확인해주세요.

    - 대상 유저가 수신할 수 있는 발송 대상 그룹으로 지정했는가?
    - 휴대전화 정보가 있는 유저가 맞는가?
    - 야간발송 금지시간(20:50 ~ 익일 8:00)에 발송하진 않았는가?
    - 메시지 유형의 규격에 어긋나는 내용은 없었는가? (제목 글자수, 본문 글자수, 이미지 크기 등)
    - url 유효성 검사에 결격 사항은 없는가? (공백이 있거나 줄바꿈이 있음, htttp:// 등 규칙에 오타가 있음 등)
    - 080 수신 거부 번호에 등록된 전화번호로 발송 시도하였는가?
  </Accordion>
  <Accordion title="브랜드 메시지 모수 계산 시 유저가 발송 대상 타겟 그룹에 포함되는지 여부도 계산하나요?">
    브랜드 메시지 모수 계산에는 유저 조건과 발송에 필요한 휴대 전화번호 정보가 있는지 여부만 확인하여 집계합니다. 발송 대상 타겟 그룹(M, N, I)에 속하는지 여부는 반영되지 않습니다.
  </Accordion>
</AccordionGroup>
