## 1. 카카오 플러스 친구 발송 프로필 등록

![노티플라이 카카오 발송 프로필 등록](/ko/user-guide/images/user-guide/kakaotalk/ko/assets/images/notifly_integration-e14894fb42f20fc1baeb7f20e176d8fb.png)

1. 노티플라이 프로젝트 '설정' 페이지의 카카오톡 채널 섹션에서 '카카오 발송 플랫폼' 옆 '**\+'** 버튼을 누르면 발송 프로필 등록 팝업이 나옵니다.
2. 개설하신 카카오톡 플러스 친구 ID(플러스 친구 검색용 ID) 및 채널 관리 권한이 있는 관리자의 연락처, 사업자 카테고리를 모두 입력해주세요.
3. 토큰 요청 버튼을 누르시면 입력하신 관리자 연락처의 카카오톡 계정으로 인증 요청이 전송 됩니다. 카카오톡으로 전송된 인증 토큰을 입력하신 후 저장하기를 눌러주시면 노티플라이 카카오채널 설정이 완료 됩니다.

## 2. 카카오 친구톡 테스트 메시지 발송

[카카오 친구톡 테스트 메시지 발송](/ko/user-guide/channel/kakaotalk/friendtalk-guide#3-5-테스트-메시지-발송) 을 통하여 발송 준비가 잘 되었는지 확인해 주세요.
