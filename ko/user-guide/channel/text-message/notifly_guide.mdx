## 1. 발신 번호 사전등록

<Note>
  전기통신사업법 제84조의2(통신사업자의 의무) 전화번호의 거짓표시 금지 및 이용자 보호에 따라, 발신번호 사전등록제가 시행되고 있습니다. 본 제도의 통신사의 스팸문자 차단 정책에 따라, 발신번호를 사전에 등록해야만 문자를 발송할 수 있도록 하는 제도입니다.
</Note>

### 1-1. 필요 서류 준비

#### 1. 통신서비스 이용증명원

<Note>
  통신서비스 이용증명원이란 이용자 본인이 사용하는 전화번호를 증명하는 서류입니다. 이용하시는 통신사에서 발급 가능하며 발급 방법은 고객 센터에 전화를 통해 발급 요청할 수 있습니다.
</Note>

<Note>
  | 통신사         | 서류명      | 연락처           |
  | ----------- | -------- | ------------- |
  | KT 고객센터     | 가입증명원    | 1588-0114     |
  | LG U+       | 가입사실 확인서 | 1544-0001     |
  | SKT         | 이용계약증명서  | 080-8282-123  |
  | 세종텔레콤       |          | 1699-1000     |
  | KCT한국케이블텔레콤 |          | 070-8188-0114 |
  | 헬로모바일KT     |          | 1855-1144     |
  | 헬로모바일SKT    |          | 070-8188-0114 |
</Note>

<Warning>
  1. 통신서비스 이용증명원 상의 명의자는 **사업자 또는 대표자**여야 합니다.
  2. 통신서비스 이용증명원에는 마스킹(숨김) 처리한 부분이 없어야 하며, 최근 **3개월 이내** 발급된 서류만 등록 가능합니다.
</Warning>

#### 2. 이용승낙서

- (주)그레이박스에서 제공하는 양식을 [다운로드](https://notifly-content.s3.ap-northeast-2.amazonaws.com/common-files/%E1%84%8C%E1%85%A5%E1%86%AB%E1%84%92%E1%85%AA%E1%84%87%E1%85%A5%E1%86%AB%E1%84%92%E1%85%A9+%E1%84%8B%E1%85%B5%E1%84%8B%E1%85%AD%E1%86%BC%E1%84%89%E1%85%B3%E1%86%BC%E1%84%82%E1%85%A1%E1%86%A8%E1%84%89%E1%85%A5+%E1%84%8B%E1%85%A3%E1%86%BC%E1%84%89%E1%85%B5%E1%86%A8.docx)하여 필요한 내용을 작성합니다.
- '계정 명의자 정보'는 '주식회사 그레이박스'로 입력해주세요.

#### 3. 사업자등록증

### 1-2. 노티플라이에 발신번호 등록하기

1. 노티플라이에 로그인 후, **설정 \> 채널별 발신 키 정보 \> 문자** 메뉴로 이동하여 등록하기 버튼을 클릭합니다.

   ![Text Message Register](/images/text-message_register.png)
2. 플랫폼을 notifly로 선택합니다.
3. 1-1에서 준비한 서류를 첨부하고, 등록하기 버튼을 클릭합니다.

![발신번호 등록](/ko/user-guide/images/user-guide/text-message/ko/assets/images/register_notifly_platform_for_text_message-c25ae13c9417fa09b113567f2f129e1b.png)

## 2. 문자 테스트 메시지 발송

캠페인 생성에서 문자 테스트 메시지 발송을 통하여 발송 준비가 잘 되었는지 확인해 주세요.

주의사항:

- 문자 메시지 발송에는 이모티콘 사용이 불가능합니다. 참고: [FAQ - 문자 메시지 이모티콘 사용 불가](/ko/user-guide/faq#1-메시지-형식)
- 문자 메시지가 스팸으로 분류되어 발송되지 않을 수 있습니다. 통신사 자체 스팸 필터링이 동작하였을 경우 휴대폰 자체의 문자 스팸함이 아닌 통신사에서 직접 관리하는 스팸 서비스 쪽에서 분류될 수 있습니다. 자세한 내용은 [통신사별 스팸차단 안내](https://www.smsko.co.kr/info_law/law_spamx.html)를 참고해 주세요.

## 3. 080 수신거부번호 설정 (선택)

광고성 메시지를 전송하기 위해서는, 문자 메시지 내용 앞 '(광고)' 라는 문자와 함께 수신거부 번호를 안내해야 합니다. 080 수신거부번호 발급이 필요한 경우, [contact@notifly.tech](mailto:contact@notifly.tech) 또는 슬랙 채널로 문의해주세요.

- 080 번호는 발급 번호당 **월 3만원 (VAT 포함)의 추가 과금**이 발생합니다. 노티플라이 월 구독료와 함께 청구됩니다.
- 080 번호 발급은 영업일 기준 3-4일 소요됩니다.
- '광고성 메시지 여부'가 체크된 문자 메시지 발신 시 080 수신거부 명단을 필터합니다.
  - 카카오 광고성 메시지(친구톡, 브랜드메시지)의 대체 문자 발송에 대해서도 해당합니다.
- 080 수신거부번호 명단은 API를 통해 고객사에서 직접 관리도 가능합니다([조회, 추가, 삭제](https://docs.notifly.tech/ko/developer-guide/http-api/http-api-guide#8-manage-text-message-080-block-service) 가이드).
  - 수신 거부 번호 등록 시 웹훅 발송도 지원합니다.

    ![080 Webhook](/images/080_webhook.png)
