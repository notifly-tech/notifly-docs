</Check>

노티플라이는 Amplitude와 두 가지 방식으로 연동할 수 있습니다.

## 연동 방식

### 1. 이벤트 스트리밍

Amplitude에서 수집되고 있는 이벤트와 유저 정보를 노티플라이에 실시간으로 전송하는 기능입니다. 별도의 개발 작업 없이 Amplitude의 이벤트 데이터를 활용하여 캠페인을 실행할 수 있습니다.

**주요 특징:**
- Amplitude에 로깅되는 이벤트를 노티플라이에 준-실시간으로 전송
- 유저 속성 정보도 함께 동기화 가능
- 월 1천만 이벤트까지 무료로 연동 (Amplitude 정책)

**사용 사례:**
- 특정 이벤트 발생 시 자동으로 푸시 알림 발송
- 유저의 행동 패턴 기반 캠페인 실행
- 실시간 이벤트 트리거 활용

[이벤트 스트리밍 설정 가이드 보기 →](/ko/user-guide/integrations/amplitude/amplitude-event-streaming)

### 2. 코호트 동기화

Amplitude에서 생성한 코호트(사용자 그룹)를 노티플라이의 사용자 그룹으로 동기화하는 기능입니다. Amplitude의 강력한 분석 기능으로 만든 타겟 그룹을 노티플라이 캠페인에 바로 활용할 수 있습니다.

**주요 특징:**
- Amplitude 코호트를 노티플라이 사용자 그룹으로 자동 동기화
- 정기적인 동기화 스케줄 설정 가능
- 코호트 변경사항 자동 반영 (추가/제거된 사용자)

**사용 사례:**
- 장바구니 이탈 사용자 대상 리마인드 캠페인
- 신규 기능 미사용자 대상 온보딩 캠페인
- 30일 이상 미접속 사용자 대상 재활성화 캠페인

[코호트 동기화 설정 가이드 보기 →](/ko/user-guide/integrations/amplitude/amplitude-cohort-sync)

## 주의사항

- 두 가지 연동 방식은 독립적으로 사용 가능하며, 필요에 따라 함께 사용할 수도 있습니다.
- 각 연동 방식은 별도의 설정이 필요합니다.

## FAQ

    네, 두 기능은 독립적으로 동작하므로 필요에 따라 둘 다 사용하실 수 있습니다. 이벤트 스트리밍으로 실시간 이벤트를 활용하고, 코호트 동기화로 타겟 그룹을 관리하는 식으로 함께 활용하시면 더욱 효과적입니다.
  </Accordion>

    - **코호트 동기화**: Amplitude의 분석 기능으로 정의한 특정 사용자 그룹을 타겟팅하는 캠페인에 적합합니다.
